# scoredle

A way to record, track, and compare friends and families wordle and worldle scores.

- created using `npm create vite@latest`

```javascript
// 0 = gray box, 1 = yellow box, 2 = green box
[
  {
    date: "Sun Feb 27 2022",
    scores: [
      {
        name: "Al",
        score: [
          [0, 0, 1, 1, 2],
          [1, 1, 1, 1, 2],
          [0, 1, 2, 1, 0],
          [2, 2, 2, 2, 2],
        ],
      },
    ],
    wordle: "#235",
  },
];
```
